# Meeting Room Manager

## Stacks:

 * Spring Web
 * Spring Data JPA
 * H2 Database
 * Java 8
 * Maven
 
## API Endpoints

* Create new meet room
POST - /api/v1/rooms

* List all meet rooms
GET - /api/v1/rooms

* Find by Id
GET - /api/v1/rooms/{id}

* Update by Id
PUT - /api/v1/rooms/{id}

* Delete by id
DELETE - /api/v1/rooms/{Id}
